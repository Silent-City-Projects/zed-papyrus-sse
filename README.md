# Papyrus SSE language support for Zed
Adds syntax highlighting and snippets for the Papyrus scripting language used in Skyrim Special Edition to the [Zed](https://zed.dev/) editor.

Makes use of the [tree-sitter-papyrus-sse](https://github.com/Silent-City-Projects/tree-sitter-papyrus-sse) project.

Supports only the SSE version of Papyrus.
